Política de Privacidade

Última atualização: Abril de 2026

A sua privacidade é importante para nós. Esta política explica quais dados coletamos, como os usamos e quais são os seus direitos.

1. Dados que coletamos

Coletamos os seguintes dados quando você se cadastra ou usa o tijolo:

  • Nome completo
  • Endereço de e-mail
  • Número de celular (opcional)
  • Nome do escritório
  • Dados de uso da plataforma (arquivos publicados, ações realizadas)

2. Como usamos seus dados

  • Para criar e gerenciar sua conta
  • Para enviar notificações relacionadas aos seus projetos
  • Para enviar comunicações sobre o tijolo (você pode cancelar a qualquer momento)
  • Para melhorar a plataforma com base no uso

3. Compartilhamento de dados

Não vendemos seus dados. Compartilhamos informações apenas com os serviços de infraestrutura que usamos para operar a plataforma (Supabase, Cloudflare, Resend), todos em conformidade com a LGPD.

4. Seus direitos (LGPD)

De acordo com a Lei Geral de Proteção de Dados (Lei 13.709/2018), você tem direito a:

  • Acessar seus dados pessoais
  • Corrigir dados incorretos
  • Solicitar a exclusão dos seus dados
  • Exportar seus dados em formato portável

Para exercer qualquer desses direitos, entre em contato: contato@tijolo.com.br

5. Retenção de dados

Seus dados são mantidos enquanto sua conta estiver ativa. Após a exclusão da conta, os dados são removidos em até 30 dias.
