Última atualização: Abril de 2026

Ao usar o tijolo, você concorda com estes termos. Leia com atenção.

1. O serviço

O tijolo é uma plataforma SaaS de gestão de projetos para escritórios de arquitetura e profissionais AEC. O acesso é fornecido mediante assinatura mensal para o Arquiteto responsável. Colaboradores externos acessam gratuitamente mediante convite.

2. Uso permitido

  • Usar a plataforma para fins profissionais de gestão de projetos AEC
  • Convidar colaboradores e clientes para projetos dos quais você é responsável

3. Uso proibido

  • Usar a plataforma para fins ilegais ou antiéticos

As assinaturas são cobradas mensalmente via cartão de crédito. Você pode cancelar a qualquer momento. O cancelamento é efetivo no final do período já pago — não há reembolso proporcional.

Você mantém todos os direitos sobre os arquivos que publica na plataforma. O tijolo não reivindica propriedade sobre seu conteúdo. Você é responsável pelo conteúdo que publica.

6. Disponibilidade

Nos esforçamos para manter a plataforma disponível, mas não garantimos 100% de uptime. Manutenções programadas serão comunicadas com antecedência.

7. Limitação de responsabilidade

O tijolo não se responsabiliza por perdas de dados causadas por uso inadequado da plataforma pelo usuário. Recomendamos manter cópias locais dos seus arquivos importantes.
